# dotfiles

kos526418's dotfiles

## Overview

- Require Vim < 8.x
- Recommended Bash < 4.x

## Setup for Git

```bash
git clone https://github.com/kos526418/dotfiles
cd dotfiles
bash setup.sh
```

UnLink

```bash
cd path/to/dotfiles
bash unlink.sh
```
